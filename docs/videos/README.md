The GIF is the README-facing preview because GitHub renders GIFs inline reliably. The WebM is kept as the higher-quality source recording. The main demo runs a recipe in Temporal and AWS Durable, then runs the shared recipe-vector scenarios in both runtimes.

## Regenerate

```bash
pnpm --filter @agentic-cookbook/tutorials-web videos:install
pnpm run demo:record
pnpm --filter @agentic-cookbook/tutorials-web videos:record
```

`pnpm run demo:record` uses `ffmpeg` to generate the README GIF when `ffmpeg` is available on `PATH`.
